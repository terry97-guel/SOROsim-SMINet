This folder contains all Linkage related functions and classes.
The Twist class, jointtwist_pre and jointtwist are used to set the 
degrees of freedom active in each piece of the link with the help of the 
custom base app.
The PlottingParameters class sets the plot options for the linkage plot 
such as camera position, axis limits, and frame rate for the dynamic 
simulation and other properties.
The folder also contains a folder containing the Linkage class and 
all it's methods.
